## Community Contributors

We appreciate all contributions, big and small! This list will grow as more people contribute to YONI.

### How to be listed here

Contributions include but are not limited to:
- Code contributions (features, bug fixes, refactoring)
- Documentation improvements
- Design work (UI/UX, graphics, branding)
- Translation work
- Bug reports and feature suggestions
- Community support and moderation
- Medical/psychological expertise and review

To be added to this list, simply make a meaningful contribution to the project.  
The list is automatically updated based on GitHub contributions, or you can submit a PR to add yourself.
